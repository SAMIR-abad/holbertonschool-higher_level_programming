The ``MyList`` class
====================

Import the class:
    >>> MyList = __import__('1-my_list').MyList

Normal case:
    >>> my_list = MyList([1, 4, 2, 3, 5])
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

Empty list:
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []
